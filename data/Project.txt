PROJECT 2: AI Interview Prep
------------------------------------------------
Description:
An AI-powered interview preparation platform that simulates job interviews by generating personalized questions from job descriptions. It evaluates user responses by comparing them with uploaded resumes using RAG.

Key Features:
- Gemini API for question generation
- RAG pipeline for resume analysis
- JWT authentication
- Resume upload with Multer
- Cloudinary file storage
- PDF parsing for extracting resume data

Tech Stack:
Node.js, Express.js, MongoDB, Tailwind CSS, JWT, BcryptJS, Multer, PDF-Parse, Cloudinary, Gemini API, RAG

Live Link:
https://ai-interview-prep-bice-eight.vercel.app/

Timeline:
Oct 2025 – Oct 2025
